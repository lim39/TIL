# 05.트랜잭션과 잠금
- MySQL의 동시성에 영향을 미치는 3가지 소개 
    - 잠금(Lock)
        - 동시성을 제어하기 위한 기능
    - 트랜잭션
        - 데이터의 정합성을 보장하기 위한 기능
        - 논리적인 작업 셋을 모두 완벽하게 처리
        - 처리하지 못할 경우에는 원 상태로 복구해서 작업의 일부만 작용되는 현상(Partial Update)이 발생하지 않게 만들어주는 기능
    - 트랜잭션의 격리수준(Isolation level)에 대한 단원
